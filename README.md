# Influence Connect — Marketing Agency

Flask web app that connects **Brands** with **Influencers**.

## Quick Start

```bash
cd influence-connect
python -m venv venv
venv\Scripts\activate          # Windows
pip install -r requirements.txt
copy .env.example .env         # add your Supabase DATABASE_URL
python run.py
```

Open **http://127.0.0.1:5000**
